# 1002 AI Sign Phrase Assistant

`1002_AI_Sign_Phrase_Assistant` 是一个面向聋哑人日常交流的离线式手语辅助沟通终端演示工程。系统沿用 STM32N647 摄像头、LCD、NPU 手部关键点模型和 USART1，在板端完成 21 个手部关键点解析、10 个常用手势短语识别、稳定保持确认、LCD 显示和 JSONL 串口输出。

目标闭环：

```text
摄像头采集
-> 板端 AI 输出 21 个手部关键点
-> 规则层识别 10 个常用手势短语
-> 1.2 秒稳定保持确认
-> LCD 显示候选短语/保持进度/确认结果
-> USART1 输出 JSONL 事件
-> 后续可接语音播报模块
```

## 串口输出

USART1 参数：`115200 8N1`。

确认成功后输出一行 JSONL：

```json
{"t":123456,"ver":1002,"event":"sign_confirmed","id":"G01","phrase":"你好","gesture":"Open Palm","hold_ms":1200,"cooldown_ms":1500}
```

电脑端可以用串口助手直接观察，也可以后续接语音模块，把 `phrase` 字段播报出来。

## 手势定义

| ID | 输出短语 | 演示手型 |
| --- | --- | --- |
| G01 | 你好 | 五指张开，拇指也张开，正对摄像头 |
| G02 | 谢谢 | OK 手势，拇指和食指捏合，中指/无名指/小指伸展 |
| G03 | 我需要帮助 | 握拳，四指收拢 |
| G04 | 我不舒服 | 拇指向下，其余手指收拢 |
| G05 | 请稍等 | 只伸食指 |
| G06 | 我想喝水 | 伸食指/中指/无名指三指 |
| G07 | 请联系家人 | 电话手，拇指和小指伸展，其余手指收拢 |
| G08 | 是的 | 拇指向上，其余手指收拢 |
| G09 | 不是 | V 手，伸食指和中指 |
| G10 | 紧急情况，请帮忙 | 四指伸展、拇指收起，区别于 G01 |

G01 和 G10 都接近张掌，演示时要刻意区分：G01 拇指张开，G10 拇指贴住掌侧或收起。
